# SpaceMission
🚀 Space Mission Data Analysis

This project explores global space missions using a dataset from Next Spaceflight, covering launches since the beginning of the Space Race in 1957.
The goal of this project is to analyse trends in space exploration and transform raw data into meaningful insights using data visualisation techniques.

📊 Project Objectives

The analysis focuses on answering key questions such as:

Which organisation launched the most missions each year?

How has the cost of space missions changed over time?

Which months are the most popular for space launches?

Have space missions become safer over time?

Which countries dominate space launches?

🛠️ Technologies Used

Python

Jupyter Notebook

pandas

Matplotlib

Plotly

📈 Visualisations Created

The project includes multiple visualisations to understand trends in space missions:

📉 Year-on-Year launch trends

📊 Organisation launch comparison

🌍 Country-wise launch distribution

📅 Monthly launch trends

📦 Sunburst charts for launch organisations

🗺️ Choropleth maps showing global launch activity

📂 Dataset

The dataset contains information such as:

Organisation responsible for the launch

Launch date

Mission status (Success / Failure)

Launch cost

Launch location

Country of origin

Source: Next Spaceflight
